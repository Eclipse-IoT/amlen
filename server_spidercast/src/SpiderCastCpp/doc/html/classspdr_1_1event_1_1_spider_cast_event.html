Detailed Description

Superclass of event hierarchy.

All SpiderCast events have a single common ancestor, the class SpiderCastEvent. The event class hierarchy reflects the functional division of events to functional units, such as error events, membership events, and so on.

Use the SpiderCastEvent::getType() method to select to correct sub-type to cast to in order to get event details.

It is recommended to use:

boost::shared_ptr<X> p_sub_type = boost::static_pointer_cast<X>(boost::shared_ptr<Y> p_ancestor);

For example:

  • SpiderCastEvent_SPtr event = ...;
  • FatalErrorEvent_SPtr fep = boost::static_pointer_cast<FatalErroeEvent>(event);
