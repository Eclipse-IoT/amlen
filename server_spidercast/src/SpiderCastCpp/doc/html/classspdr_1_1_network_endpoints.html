Detailed Description

Represents node network end-points, assuming multiple interfaces and a single port.

Different interfaces may reside on different networks without connectivity. Network scope is identified by the Network-ID; Two Nodes that have host-addresses with the same network-ID can communicate with each other.

This class holds a vector of scoped-addresses, and a port number. Each scoped-address is a pair holding two strings; first is the IP address or host-name, second is the Network-ID.

Order within the vector does matter - the scoped addresses are usually scanned from beginning to end when establishing a connection.

This class supports equality operators == and !=, as well as less than comparisons and all its variations <, <=, >, >=.
