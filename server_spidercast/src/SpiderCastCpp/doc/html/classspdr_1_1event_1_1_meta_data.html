Detailed Description

Holds the node's meta-data.

The node's meta-data is composed of

  • The AttributeMap - a map holding key-value attributes.
  • The incarnation number - an integer identifier that increases every time a node starts.

The meta-data can be compared for equality using operators == and !=. Equality means that all the elements of the meta-data are equal - deep compare.

Copy constructor and assignment operator are default synthesized (shallow copy). For deep copy use clone().
