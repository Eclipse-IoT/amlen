Detailed Description

A fatal error event.

Event type: EventType::Fatal_Error.

This event is delivered when a SpiderCast encounters a fatal error. The SpiderCast instance closes down internally and frees all resources. The state of the instance after this event is SpiderCast::NodeState::Error.
