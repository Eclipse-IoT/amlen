Detailed Description

An attribute map.

The AttributeMap is a std::map with String keys and AttributeValue values. In addition, it supports conversion to a string representation with toString(), and supports equality operators == and !=.

The attribute map is included in the MetaData, which is delivered in some MembershipEvent sub-types, as part of the MembershipService.
