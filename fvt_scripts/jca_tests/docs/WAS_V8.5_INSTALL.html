WebSphere Application Server V8.5 Installation Instructions

  • Install WAS V8.5 on p-linux
  • Install WAS V8.5 on linux

Install WAS V8.5 on p-linux (Based on mar092)

Setup

  1. cd /WAS/installables/WAS85
  2. mkdir DISKS
  3. mkdir IM
  4. unzip InstalMgr1.5.2_LNX_PPC_WAS_8.5.zip -d IM/
  5. unzip WASV8.5_*_OF_3.zip -d DISKS/

Installation Manager V1.5.2

  1. /WAS/installables/WAS85/IM/consoleinst.sh
  2. Follow the prompting to install Installation Manager v1.5.2
NOTE: Change the installation directory to /opt/IBM/InstallationManager.1.5.2 to avoid conflicts with other versions. It will also create /opt/IBM/IMShared and /var/ibm/InstallationManager

WebSphere Application Server V8.5

  • Edit /WAS/installables/WAS85/responsefile85.xml with the location of the repository.config file in the DISKS directory, and the directory paths to install WAS into.
  • responsefile85.xml is already set up based on installation for mar092.
NOTE: Change the installation and eclipse directories to /opt/IBM/WebSphere85/AppServer to avoid conflicts with other versions.
  1. cd /opt/IBM/InstallationManager.1.5.2/eclipse:
  2. ./IBMIM -showProgress -silent -input /WAS/installables/WAS85/responsefile85.xml -acceptLicense

Post Installation

  • Once WAS V8.5 is installed, create the WAS profile by running:
    /opt/IBM/WebSphere85/AppServer/bin/manageprofiles.sh -create -profilePath /opt/IBM/WebSphere85/AppServer/profiles/AppSrv01 -templatePath /opt/IBM/WebSphere85/AppServer/profileTemplates/default/ -profileName AppSrv01
  • Start the server:
    /opt/IBM/WebSphere85/AppServer/bin/startServer.sh server1 -profileName AppSrv01
  • Setup security:
    1. Log on to the admin console at http://10.10.10.10:9060/ibm/console
    2. Go to Security - Global Security
    3. Run the Security Configuration Wizard
    4. Select Enable Application Security, click next
    5. Select Federated Repositories, click next
    6. Enter username/password: admin/admin
    7. Click Finish
    8. Save changes

Install WAS V8.5 on linux (Based on mar215)

Setup

  1. cd /installables/WAS85
  2. unzip RSA_WS_8.5.1_Setup.zip
  3. unzip RAD_WS_8.5.1_TE_8.5.0.1_1.zip
  4. unzip RAD_WS_8.5.1_TE_8.5.0.1_2.zip

WebSphere Application Server V8.5 + Installation Manager V1.5.2

Installation is done through the Rational Software Architect V8.5.1 Installation Manager repository. This will install both the installation manager and WAS.

  1. Run: /installables/WAS85/RSA4WS_SETUP/imLauncherLinux.sh
  2. If the installation manager does not launch, run: /installables/WAS85/RSA4WS_SETUP/InstallerImage_linux_gtk_x86/install
  3. Select only WebSphere Application Server V8.5 from the list of installable items.
  4. Follow the prompting to install Installation Manager v1.5.2
NOTE: Change the installation directory to /opt/IBM/InstallationManager.1.5.2 to avoid conflicts with other versions. It will also create /opt/IBM/IMShared and /var/ibm/InstallationManager
NOTE: Change the installation and eclipse directories to /opt/IBM/WebSphere85/AppServer to avoid conflicts with other versions.

Post Installation

Once installation is complete, the installation manager will launch the WAS Profile Management tool. Create a new Application Server profile using default settings, and admin/admin for username/password.

Once the profile is created, first steps should appear. Start the server.
